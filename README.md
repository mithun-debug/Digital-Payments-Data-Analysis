# Digital-Payments-Data-Analysis (2018 - 2024)
## Project Overview
This project involves a comprehensive analysis of PhonePe's transaction and demographic data across various states and districts in India. The goal is to uncover insights into transaction trends, user behavior, and device usage patterns, and to correlate these with regional demographics using Python and data visualization libraries.
## Dataset used
- <a href= "https://github.com/mithun-debug/Digital-Payments-Data-Analysis/blob/main/Phonepe_Dataset.xlsx"> Dataset</a>
## Tech Stack
Language: Python
Libraries: * Pandas & NumPy for data cleaning and manipulation
Matplotlib & Seaborn for static visualizations
## Analysis Workflow
- Data Cleaning & Quality Checks
  Handled missing values and verified data types.
  Validated if the sum of district-level transactions matched the reported state-level totals to ensure data integrity.

- Exploratory Data Analysis (EDA)
  Tracking the growth of digital payments over the years and quarters.
  Transaction Types: Identifying which categories (e.g., Recharge vs. P2P) dominate specific regions.
  Brand Loyalty: Analyzing which mobile brands are most prevalent among digital payment users.

- Advanced Insights
  Calculated the ratio of registered users to the total population.
  Analyzed how population density impacts transaction volume using Pearson correlation.
  Determined the Average Transaction Value (ATV) across different states to identify spending habits.
## Business Insights
- Maharashtra and Karnataka consistently rank in the top 5 for transaction volume.
- There is a strong positive correlation between population density and the number of app opens.
- Xiaomi and Samsung remain the dominant device brands among PhonePe users.
- Data suggests opportunities for targeted expansion in emerging districts.
- Significant growth observed in digital transactions post-2020, indicating accelerated fintech adoption.
